# Parent Question → Little Lab Coats Map

Use this to turn recurring parent questions into posts that naturally bridge into a live LLC page.

## Important note
Many LLC paid bundles exist in the catalog, but some checkout/store paths are still marked `pending` in the product catalog. So for social right now, the safest path is:
1. answer the question well
2. point to a live free lesson/resource
3. then mention the related bundle or full-grade path more softly

## Best-performing CTA styles to test

### Save-style CTAs
- Save this for next week’s science block.
- Save this for your next rainy day.
- Save this if science planning sneaks up on you.

### Comment-style CTAs
- Comment `BOOKS` and I’ll make the printable list.
- Comment `WEATHER` if you want the weather lesson.
- Comment `ANIMALS` if you want more animal science ideas.

### LLC click-through CTAs
- I have a free version of this on Little Lab Coats.
- If you want this ready to use, the printable is on the site.
- If you want the full lesson done for you, start with the free sample on LLC.

## Recommended content-to-offer ladder
1. **Question post** — answer a real parent pain point
2. **Freebie post** — point to the matching free lesson/resource
3. **Authority post** — explain Serena’s approach to science at home
4. **Offer post** — soft mention of unit bundles/full-grade bundles

That sequence should feel helpful instead of salesy, which is exactly what this audience needs.
