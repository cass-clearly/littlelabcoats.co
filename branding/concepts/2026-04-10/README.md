# Little Lab Coats branding concept pack additions — 2026-04-10

Review-only branding add-on pack for Serena. No live homepage changes made.

## New options

### Option E — Cartoon Lab Coat
- Serena-requested direction
- Simple cartoon lab coat mark
- Uses the brighter, playful color feeling from prior Option B
- Keeps the simplicity/icon-friendliness closer to prior Option A
- Best if we want the logo to tie very literally to the name without getting stiff

### Option F — Spark Star Badge
- Bright celebratory spark/star mark with notebook detail lines
- Feels fun, energetic, and recognizable in small sizes
- Best if we want a broader curiosity/discovery signal instead of a literal lab object

### Option G — Pocket Microscope
- Simplified microscope icon with rounded kid-friendly styling
- Strongest classic science cue among the new options
- Best if we want immediate educational/science recognition with a softer feel

### Option H — Curious Owl Coat
- Minimal owl + lab coat hybrid mark
- Warmest and most mascot-like of the new options, but still simple
- Best if we want something especially memorable and family-facing

## Notes
- All four options include the full `Little Lab Coats` name in the header treatment

- No live homepage changes were made
