✍️ Short Response — Use Evidence

Explain how one redesign improved a bridge model.

Prompt: Name the bridge problem, describe one change from version 1 to version 2, and explain what test evidence makes the new version more trustworthy.

👩‍🏫 Parent/Teacher Notes

  • Use the quiz and short response together: Multiple choice checks recognition of the bridge-building process, while the short response checks reasoning with evidence.
  • Look for challenge-specific language: Strong answers mention the gap, shape choice, fair testing, weak point, redesign, version 1, and version 2.
  • Fast grading tip: Scan for the words gap, shape, fair test, weak point, redesign, evidence, version 1, and version 2.

✂️ PARENT ANSWER KEY — Detach before giving worksheet to students

Parent note: Student wording may vary if the engineering reasoning is correct.

Multiple-Choice Answer Key

  1. It names the user, gap, and success rules
  2. Keeping the same gap and same type of weight
  3. The fold changes the shape and can make it stronger
  4. To explain what each part is supposed to do
  5. The first bridge model built from the plan
  6. Sagging, slipping, or how much weight it holds
  7. So you can tell which change helped
  8. It includes the problem, the improvement, and test evidence

Short Response — What Success Looks Like

  • Student names the bridge problem clearly.
  • Student describes one specific redesign change.
  • Student cites test evidence showing why version 2 is stronger.

Short Response — Sample Acceptable Answer

“My problem was helping a toy animal cross a gap safely. I changed version 2 by adding folded sides because version 1 sagged in the middle. The evidence is that version 2 held more pennies before bending and stayed flatter across the same gap.”
