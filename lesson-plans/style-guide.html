✍️ Little Lab Coats Style Guide

The Editorial Bible for NGSS-Aligned Science Curriculum
Kindergarten through 5th Grade

Created by Margaret Hoffman · Last Updated: February 2026

Purpose: This document establishes the editorial standards for all Little Lab Coats curriculum materials. Every lesson plan, worksheet, parent guide, and supplementary material must conform to these rules. Consistency is not optional — it is the foundation of professional educational publishing.

Why this matters: Parents who purchase curriculum are making a significant investment in their children's education. Inconsistent terminology, erratic formatting, and sloppy punctuation signal carelessness. We cannot afford to look careless. Every typo is a crack in our credibility.

⚠️ Non-Negotiable Rule: If a decision is documented in this style guide, it applies to every future lesson without exception. Deviations require explicit approval and documentation of the change. Style drift kills brands.

Brand Identity

Official Brand Name

CANONICAL: Little Lab Coats (three words, title case)
Never: "Little Labcoats," "little lab coats," "LittleLabCoats," "LLC" (conflicts with Limited Liability Company)

Tagline

Primary: "NGSS-Aligned, Hands-On Science for Homeschool Families"

Short version: "Real Science, Real Experiments, Real Fun"

Target Audience

  • Primary: Homeschool parents educating multiple children (typically ages 5–11, K–5th grade)
  • Secondary: Micro-schools, homeschool co-ops, afterschool enrichment programs
  • Geographic: United States (NGSS alignment is US-specific)
  • Values: Secular, science-based, inquiry-driven, minimal screen time

Brand Voice

For parents/teachers: Warm but authoritative. Written by a scientist who is ALSO a homeschool parent. We understand the chaos, the mess, the reality of teaching multiple ages simultaneously. Practical, not preachy. Evidence-based, not dogmatic.

For students: Enthusiastic and curious. We use exclamation marks (sparingly) to signal excitement. Questions are genuine, not rhetorical. We assume kids are smart and capable — we scaffold complexity, we don't dumb it down.

Tone checklist: Before publishing, ask:
  • Does this sound like a real teacher talking to a real parent?
  • Would I say this out loud to a friend?
  • Is the science accurate without being condescending?
  • Does it respect the parent's time and intelligence?

Document Structure

Required Sections (in exact order)

Every lesson plan MUST contain these sections in this sequence:

  1. Title — Emoji + Lesson name + Emoji (e.g., "🎢 Marble Run Madness 🎢")
  2. Subtitle — Grade range or unit designation in italics
  3. Metadata Bar — Grade, duration, subject, optional kit designation
  4. Hero Image or Emoji — Visual anchor with caption
  5. NGSS Standards Box — Standards alignment in purple box
  6. The Big Idea — 2–3 sentences summarizing the lesson's core concept
  7. The Science — Age-appropriate science theory, concepts, and background that introduces students to the WHY behind the activity. This section eases kids into the lesson by building foundational understanding before they start the practical work. Should include key concepts explained in kid-friendly language.
  8. Key Vocabulary (Tip Box) — 6–10 terms introduced in this lesson
  9. Materials — Organized by source (kit vs household) with subcategories
  10. Let's Get Started — Numbered steps in blue boxes with strong headers
  11. Discussion Questions — In green boxes, 5–8 questions
  12. Extensions — Optional enrichment activities in purple box
  13. Parent/Teacher Notes — Gray dashed box with practical tips

Discussion Questions: Suggested Answer Hints

RULE: Every discussion question must include a suggested answer directly beneath it, formatted in italics. This helps guide the instructor during live discussion without needing to flip to the answer key.

Format:
  • Question text in normal weight
  • Suggested answer immediately below in <em> italics, using the CSS class .discussion-hint (styled with muted color, slightly smaller font)
  • Answers should be conversational and acknowledge multiple valid responses where applicable
  • Use phrasing like "Look for answers like…" or "They might say…" to keep it open-ended
  • Answers must be scientifically accurate and age-appropriate for the target grade level
CSS class:
.discussion-hint { font-style: italic; color: #666; font-size: 0.95em; margin-top: 2px; }
✅ Correct

What would happen to the reef if all the parrotfish disappeared?
Look for: The coral would get overgrown with algae because parrotfish eat the algae that competes with coral for space. Without them, algae would smother the reef.

❌ Wrong

What would happen to the reef if all the parrotfish disappeared?
(No suggested answer provided — instructor must flip to the answer key)

Required Sections (continued)

  1. Student Worksheet(s) — Page break, then student-facing materials
  2. Parent Answer Key — Final page, clearly marked for removal
  3. Remarq Script Tag — HTML footer script for feedback layer
INCONSISTENCY FOUND: Some lessons include a "Setting:" field in the metadata bar (e.g., "Homeschool (2 students)"). Others omit it.

RESOLUTION: Never include a "Setting:" field. The curriculum is designed for homeschools by default. Omit this field entirely.

Optional Sections (where applicable)

  • Safety Box — Orange box warning about specific hazards (hot glue, sharp objects, choking hazards)
  • Setup Instructions — If significant prep is required, add a "Setup (5 min)" step before Procedure Step 1
  • Grade-Differentiated Sections — For multi-grade lessons (3rd & 5th), use colored boxes to separate age-appropriate content

Typography Rules

Heading Hierarchy

Level Tag Usage Capitalization Example
H1 <h1> Lesson title only Title Case The Great Push & Pull Sort
H2 <h2> Major sections Title Case Materials, Procedure, Discussion Questions
H3 <h3> Subsections within H2 Title Case From Your Science Kit, For Older Siblings
H4 <h4> Rare; use for tertiary divisions Title Case Optional: Additional Context

Number Formatting

RULE: Spell out numbers one through nine. Use numerals for 10 and above.
EXCEPTION: Always use numerals for:
  • Measurements: 5 feet, 2 cups, 3 inches
  • Ages: 5 years old, 8-year-old child
  • Quantities in lists: 3 magnets, 4 wheels, 10 paperclips
  • Step numbers: Step 1, Step 2
  • Grades: 3rd grade, 5th grade
✅ Correct

Test three different surfaces.

The car rolled 12 inches.

You need 5 paperclips.

This works for 5-year-olds.

❌ Wrong

Test 3 different surfaces.
(spell out small numbers in prose)

The car rolled twelve inches.
(use numerals with units)

You need five paperclips.
(list quantities = numerals)

Range Formatting

RULE: Use en dash (–) for all ranges, with NO spaces.
Examples: 30–45 minutes, 3rd–5th grade, pages 12–18, 0°F–32°F
COMMON ERRORS:
  • 30-45 minutes — That's a hyphen, not an en dash
  • 30 – 45 minutes — No spaces around en dash in ranges
  • 30 to 45 minutes — Use en dash, not the word "to"

Emphasis Conventions

  • Bold: Key terms on first use, step headers, procedural emphasis
    Example: "A push moves something away from you."
  • Italic: Book titles, introducing scientific terms, emphasis in dialogue
    Example: Read Forces Make Things Move by Kimberly Bradley.
  • ALL CAPS: Rare. Only for strong student emphasis in questions or warnings
    Example: "ONLY use the beak — no hands!"

Font Stack

Body text: Georgia, 'Times New Roman', serif (professional, readable, print-friendly)
Headings: 'Nunito' or 'Quicksand' (web fonts for visual warmth), fallback to sans-serif
Code/technical: Consolas, Monaco, monospace (rare, only for HTML examples)

Punctuation Standards

Serial (Oxford) Comma

POLICY: Always use the serial comma in lists of three or more items.
RATIONALE: Eliminates ambiguity, aligns with Chicago Manual of Style (educational publishing standard)
✅ Correct

You need a ball, a ramp, and a ruler.

Test with ice, salt, and water.

❌ Wrong

You need a ball, a ramp and a ruler.

Test with ice, salt and water.

Contractions

POLICY: Contractions are permitted and encouraged in student-facing text and parent notes to maintain conversational tone.
EXCEPTIONS: Avoid contractions in formal sections (Standards box, academic discussion)

Permitted: you'll, we'll, it's (it is), doesn't, can't, that's, what's, let's

Forbidden: ain't, gonna, wanna, coulda/shoulda/woulda

Exclamation Marks

POLICY: Use sparingly to signal genuine excitement or surprise.
LIMIT: No more than 3 exclamation marks per page of body text.
NEVER: Multiple exclamation marks (!!!) or mixed punctuation (!?!)
✅ Good Use

"Whoa! What happened?"

"You just made ice cream!"

Watch it spin!

❌ Overuse

"This is so cool!!!"

"Try it! Test it! Measure it!"
(three in a row = overkill)

List Punctuation

RULE: If list items are complete sentences, end with periods. If fragments, no periods.
CONSISTENCY: All items in a single list must follow the same rule — don't mix.
✅ Consistent

Complete sentences:

  • The ball rolled 5 feet.
  • The ramp was too steep.
  • Try it again with less height.

Fragments:

  • 1 ball
  • 2 ramps
  • 3 books
❌ Inconsistent
  • The ball rolled 5 feet.
  • Ramp too steep
  • Try again with less height
(mixing sentences and fragments)

Quotation Marks

STYLE: American style — periods and commas go INSIDE closing quotation marks, even when illogical.
EXCEPTION: Question marks and exclamation marks follow logic (inside if part of quote, outside if not)
✅ Correct

Ask: "What is a push?"

The card says "magnetic."

Did you say "pull"?

❌ Wrong

The card says "magnetic".

Ask: "What is a push"?

Terminology Glossary

Use these exact terms consistently. Do not invent synonyms.

Term Usage & Notes
Student Worksheet ALWAYS use this exact phrase for student-facing recording sheets. Never "data sheet," "recording page," "observation sheet," or "handout."
Parent Answer Key Not "Answer Key for Parents," not "Teacher Guide." Always "Parent Answer Key" with the detach instruction.
Let's Get Started The main instruction section. Never "Procedure," "Setup," "Instructions," "How to Do It," or "Directions." If pre-lesson setup is needed, call it "Setup (5 min)" as a separate box before Step 1.
The Big Idea Section introducing the core concept. Title case, no colon. Never "Big Idea:," "Concept:," or "Overview:"
Discussion Questions Never "Reflection Questions," "Think About It," or "Debrief." Consistent across all lessons.
Extensions Never "Extension Activities," "Going Further," "Challenge," or "Enrichment." Just "Extensions."
Parent Notes For kindergarten lessons. For 3rd–5th, use "Parent / Teacher Notes" or "Teacher Notes." Never "Tips," "FYI," or "Notes to Instructors."
Materials Never "Supplies," "Stuff You Need," "Shopping List," or "What to Gather." Section title is always just "Materials."
Kit Lesson When a lesson benefits from the physical kit, badge it as "🎒 Kit Lesson" in the metadata bar. Do not use "Requires Kit" or "Kit Required."
NGSS Standard Use singular "Standard:" even when listing multiple standards. This matches the established pattern. Never "Standards:" plural in the label.
Why this matters: If Lesson 1 calls it a "recording sheet" and Lesson 3 calls it a "data sheet," parents will wonder if they're different documents. They're not — we're just being sloppy. Pick one term and enforce it everywhere.

NGSS Citation Format

Standard Code Structure

FORMAT: Grade-DisciplineCode-Number
EXAMPLE: K-PS2-1, 3-LS4-2, 5-PS1-3
NEVER: Add "NGSS" prefix — it's redundant (e.g., don't write "NGSS K-PS2-1")

Standards Box Heading

SINGULAR: "NGSS Standard:" (not "Standards:")
RATIONALE: Established pattern in existing lessons. Maintain consistency even when listing multiple standards.
✅ Correct Format

📋 NGSS Standard:
K-PS2-1 — Plan and conduct an investigation to compare the effects of different strengths or different directions of pushes and pulls on the motion of an object

3-LS4-2 — Use evidence to construct an explanation for how the variations in characteristics among individuals of the same species may provide advantages in surviving

❌ Wrong Format

NGSS Standards:
(should be singular)

NGSS K-PS2-1
(don't add NGSS prefix to code)

K-PS2.1
(use hyphen, not period)

Full Standard Text

After the standard code, include the full performance expectation text. Use an em dash (—) to separate code from text.

Placement

Standards box appears immediately after the hero image/emoji section and before "The Big Idea." Always in a purple/lavender box (#f3e5f5 background, #ce93d8 border).

Materials Section Format

Standard Subcategories (in order)

  1. From Your Science Kit — Items included in the physical kit (if applicable)
  2. Household Items to Gather — Common items parents already own or can easily obtain
  3. Optional / Nice to Have — Enhances the lesson but not required
  4. Printables — PDFs to download and print (worksheets, data sheets, cards)

List Formatting

RULE: Use bullet lists, not numbered lists. Include quantities and brief descriptions.
FORMAT: [Quantity] [Item] [(optional clarification)]
✅ Clear & Specific
  • 2 bar magnets (refrigerator magnets work)
  • 10+ paperclips (metal, not plastic-coated)
  • 1 marble per child
  • Masking tape (at least 12 inches)
❌ Vague
  • Magnets
  • Paperclips
  • Marbles
  • Tape
(How many? What kind?)

Kit Designation

If a lesson includes 🎒 Kit Lesson in the metadata bar, the "From Your Science Kit" section is mandatory and should list 3–8 specialty items that justify the kit.

Subsection Styling

Materials section uses a yellow/gold box (#fff8e1 background, #ffcc80 border). Subsection headings (<h3>) inside the box use orange color (#e65100).

Emoji Usage Policy

When Emoji Are Appropriate

  • Titles: Flanking the lesson title (🧲 Magnet Magic 🧲)
  • Metadata bar: One emoji per field (📚 Grade, ⏱️ Duration)
  • Section icons: Optional emoji before section titles (🎯 The Big Idea, 📦 Materials)
  • Kindergarten lessons: Liberally throughout to maintain engagement
  • Student worksheets: As visual cues for sorting, labeling, and matching activities

When Emoji Are NOT Appropriate

  • NGSS Standards box — formal, no emoji
  • Parent Answer Key — professional tone, no emoji
  • Mid-sentence decorative emoji — distracting
  • Replacing words (❤️ instead of "love") — accessibility issue for screen readers

Grade-Level Emoji Density

Kindergarten: High emoji use — they're pre-readers, visual cues help navigation
3rd–5th Grades: Moderate use — emoji for organization and emphasis, not decoration

Emoji Glossary (Standardized Meanings)

Emoji Meaning Usage
🎒 Kit Lesson Appears in metadata bar when lesson uses physical kit materials
🏠 Household Items Subsection header in Materials section
🧰 Supplies / Tools General equipment section in Materials
📋 Standards / Printables NGSS box header, or printable materials list
🎯 Goal / Big Idea Marks the core concept section
💡 Tip / Insight Parent tips, helpful hints, key insights
⚠️ Warning / Safety Safety notes, important cautions
✅ Correct / Success Example boxes showing correct usage
❌ Incorrect / Error Example boxes showing what to avoid
🚀 Extension / Challenge Extensions section icon

CSS & Design Standards

Page Layout

Max width: 8.5 inches (letter size)
Padding: 0.5in on screen, 0.25in in print
Font size: 11pt body text for print-friendly reading
Line height: 1.4–1.65 for readability

Box Styling Standards

  • Border radius: 6–12px (rounded corners, friendly feel)
  • Padding: 10–15px inside boxes
  • Margin: 10–20px between sections
  • Border width: 1–2px for standard boxes, 3px for emphasis boxes

Color Accessibility

All text on colored backgrounds must meet WCAG AA contrast ratio (4.5:1 minimum). Never use light text on light backgrounds or dark text on dark backgrounds.

Print Color Preservation: Always include this CSS in every lesson: 
* {
  -webkit-print-color-adjust: exact;
  print-color-adjust: exact;
}
This ensures colored boxes print correctly and aren't converted to grayscale.

Responsive Design

Include a mobile breakpoint at 640px that adjusts font size, removes multi-column layouts, and increases touch target sizes for parent phones/tablets.

Print Formatting Rules

Page Break Strategy

RULE: Student worksheets and answer keys MUST start on a new page.
METHOD: Use .page-break { page-break-before: always; } class
NEVER: Let a worksheet start at the bottom of the lesson plan page

CRITICAL: When printed, no section should be split across pages in a way that breaks readability. Use page-break-before: auto and page-break-inside: avoid on section containers. Student Worksheets and Answer Keys MUST start on new pages.

Avoid Orphans and Widows

  • Apply page-break-inside: avoid; to procedure steps, question boxes, materials boxes, discussion boxes, and all important sections
  • Apply break-after: avoid; to all headings so they don't separate from their content
  • Use orphans: 3; and widows: 3; in body text to prevent single lines stranded on pages
  • EVERY major section should start with enough breathing room — use consistent margins and avoid cramped layouts

Detachable Components

Student worksheets and answer keys are designed to be printed separately. Include clear visual separation (dashed borders, "✂️ Detach Here" instructions) on the Parent Answer Key.

Standard detach instruction:
"✂️ PARENT ANSWER KEY — Detach before giving worksheet to students"

Print CSS Block

Every lesson MUST include this print media query:

@media print {
  body { padding: 0.25in; font-size: 10pt; }
  h2 { break-after: avoid; }
  h3 { break-after: avoid; }
  .procedure-step, .question-box, .teacher-note {
    break-inside: avoid;
    page-break-inside: avoid;
  }
  * {
    -webkit-print-color-adjust: exact;
    print-color-adjust: exact;
  }
}

Answer Key Format

Required Elements

  1. Detach instruction: "✂️ PARENT ANSWER KEY — Detach before giving worksheet to students"
  2. Expected results: What data/observations should students record
  3. Sample answers: Model answers for discussion questions
  4. Troubleshooting section: Common problems and how to fix them
  5. Success criteria: What mastery looks like ("Can explain X," "Correctly sorted Y")

Styling

Container: Gray dashed border (#999), light gray background (#f5f5f5)
Heading: Gray H2 background (#757575), white text
Font size: 10pt (slightly smaller than lesson body to fit more on page)

Tone

Answer keys are written for parents, not students. Use a supportive, practical tone. Acknowledge variability ("Results may vary slightly," "If your child said X, that's also correct because…").

✅ Supportive Tone

Q: Why did the car stop?
Answer: Friction slowed it down until it stopped. If your child said "it ran out of push" or "the balloon was empty," that's also correct — they're describing the same phenomenon in their own words.

❌ Rigid Tone

Q: Why did the car stop?
Answer: Friction. If they didn't say friction, they got it wrong.

Pre-Publication Checklist

Before any lesson is published or sent to a customer, verify every item on this checklist.

  • Title includes flanking emoji and matches capitalization rules
  • Metadata bar includes all required fields in correct order (Grade, Duration, Subject)
  • No "Setting:" field in metadata (REMOVED per style guide)
  • NGSS Standards box uses singular "NGSS Standard:" label
  • Standard codes use hyphen format (K-PS2-1) not period or other separator
  • Duration uses en dash (30–45 minutes) not hyphen (30-45)
  • Grade ranges use en dash for contiguous (3rd–5th) or ampersand for non-contiguous (3rd & 5th)
  • Serial commas used in all lists of three or more items
  • Exclamation marks limited to 3 per page maximum
  • Numbers 1–9 spelled out in prose, 10+ as numerals (except measurements, ages, steps)
  • Student-facing document labeled "Student Worksheet" not "recording sheet," "data sheet," or other variant
  • Parent-facing answer key labeled "Parent Answer Key" with detach instruction
  • Main instruction section titled "Let's Get Started" not "Procedure," "Setup," or "Instructions"
  • All section titles use consistent terminology from glossary (Materials, Discussion Questions, Extensions)
  • Let's Get Started steps use consistent formatting (numbered, in colored boxes with bold headers)
  • Color coding matches grade level and subject (blue for K, green for life science, etc.)
  • Box colors match functional standards (purple for NGSS, gold for materials, etc.)
  • Multi-grade lessons use orange for 3rd grade, purple for 5th grade differentiation
  • Emoji usage appropriate for grade level (high for K, moderate for 3–5)
  • Print CSS includes color preservation and page-break rules
  • Student worksheet starts on new page (page-break-before: always)
  • Answer key starts on new page and includes "✂️ Detach" instruction
  • All internal cross-references verified (if mentioning another lesson, check lesson number is correct)
  • Spell-checked (no typos in body text, headings, or worksheets)
  • Remarq script tag present at bottom with correct document ID
⚠️ Zero-Tolerance Errors: These errors are unacceptable in published materials:
  • Misspelled student-facing text
  • Incorrect NGSS standard code
  • Wrong lesson number in cross-references
  • Broken page layout (orphaned sections, cut-off boxes)
  • Missing answer key
If you find any of these in a published lesson, pull it immediately and issue a corrected version.
