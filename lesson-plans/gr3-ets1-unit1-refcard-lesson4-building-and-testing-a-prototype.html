🛠️ Building and Testing a Prototype

Grade 3 Engineering — Quick Reference Card
Big Idea

A prototype helps engineers collect evidence about what works and what needs to improve.

Key Vocabulary
  • prototype: an early model used for testing
  • fair test: a test where important conditions stay the same
  • variable: something that can change in a test
  • failure point: the part that did not work well
  • observation: something noticed during the test
Remember This
  • Testing gives engineers evidence instead of guesses.
  • A fair test keeps the drop height, object, and test surface consistent.
  • Failure points are helpful because they show what needs redesign.
Use It to Explain
  • Start with: In my test, I observed...
  • Add: The failure point was...
  • Finish with: This evidence matters because...
