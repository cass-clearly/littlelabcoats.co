✍️ Short Response — Use Evidence

Explain how one redesign improved a fragile-object protection challenge.

Prompt: Name the problem, describe one change from version 1 to version 2, and explain what test evidence makes the new version more trustworthy.

👩‍🏫 Parent/Teacher Notes

  • Use the quiz and short response together: Multiple choice checks recognition of the design process, while the short response checks reasoning with evidence.
  • Look for challenge-specific language: Strong answers mention fragile-object protection, criteria, constraints, fair tests, failure points, or redesign evidence.
  • Fast grading tip: Scan for the words criteria, constraint, prototype, fair test, failure point, version 1, and version 2.

✂️ PARENT ANSWER KEY — Detach before giving worksheet to students

Parent note: Student wording may vary if the engineering reasoning is correct.

Multiple-Choice Answer Key

  1. It names the need and what success should look like
  2. A limit the design must follow
  3. To choose the idea that best fits the criteria and constraints
  4. The drop height and object stay the same
  5. The weakest part that did not work well
  6. It should connect to evidence from version 1
  7. The problem, solution, criteria, constraints, and evidence
  8. Because it includes specific test or redesign evidence

Short Response — What Success Looks Like

  • Student names the challenge or problem clearly.
  • Student describes one specific redesign change.
  • Student cites test evidence showing why version 2 is stronger.

Short Response — Sample Acceptable Answer

“My problem was protecting a fragile egg during a drop. I changed version 2 by adding stronger side supports and extra padding where version 1 bent. The evidence is that version 2 held its shape better and the egg moved less during the same drop test.”
