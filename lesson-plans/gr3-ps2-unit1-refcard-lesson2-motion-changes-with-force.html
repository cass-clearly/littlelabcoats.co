🚗 Motion Changes with Force

Grade 3 Physical Science — Quick Reference Card
Big Idea

A stronger push can change motion more than a weaker push. Repeated tests help us find the pattern.

Key Vocabulary
  • Force: a push or a pull
  • Motion: how something moves
  • Distance: how far something travels
  • Evidence: observations or measurements from a test
  • Compare: look at how results are the same or different
Remember This
  • Keep the object and starting line the same for a fair test.
  • Repeat the test so you have better evidence.
  • If a stronger push usually moves the object farther, that is a pattern you can explain.
