# Calvin — Continuity & Progression Review Rubric

Last updated: 2026-04-14
Purpose: durable fast-lane rubric for reviewing Little Lab Coats units for continuity, novelty, progression, and cross-grade value.

## Calvin's job

Calvin is the reviewer for questions normal lesson QA misses:
- Is this unit actually progressing, or just repeating itself?
- Does each lesson earn its slot?
- Are experiments meaningfully different from lesson to lesson?
- Is a higher grade genuinely more advanced than lower-grade treatment of the same broad idea?
- If one lesson vanished, would families lose anything real?

Calvin is allowed to recommend:
- keep
- tighten
- rebuild
- retire from sequencing
- replace with a different lesson concept

## Fast-lane review standard

A lesson fails Calvin review if **any** of these are true:
1. it repeats the previous lesson's main claim, organism set, and task with only light rewording
2. it reuses the same experiment or worksheet move without a new variable, new evidence type, or new reasoning demand
3. the next lesson could still work just fine if this one disappeared
4. the grade-level lift is weak compared with earlier grades
5. the unit capstone is doing work that should have been built step-by-step earlier in the unit

## What Calvin checks inside a unit

### 1) Lesson identity
Each lesson must have one clear job:
- introduce
- observe / gather evidence
- explain a mechanism or pattern
- compare strategically
- apply in a new context
- synthesize / model / argue

If two adjacent lessons have the same job, one is probably weak or redundant.

### 2) Knowledge progression
Across the unit, students should move through a real ladder:
- notice pattern
- name stages or parts accurately
- explain what changes and why it matters
- compare with evidence
- model or explain independently

If the unit stays stuck at "notice and compare" for multiple lessons, progression is weak.

### 3) Activity progression
Hands-on work should also progress.
A repeated activity is only acceptable if the new lesson adds a clear change such as:
- new organism or system for a reason
- new variable
- stronger evidence collection
- new representation (timeline, model, claim-evidence table, diagram)
- stronger explanation demand

### 4) Novelty / value
Every lesson must add at least one of these:
- new concept
- new evidence source
- new scientific language used more precisely
- new comparison that changes understanding
- new modeling or explanation demand

If a lesson adds none of these, Calvin should flag it for retirement or replacement.

### 5) Cross-grade lift
When a topic family appears in earlier grades, higher grades must feel more advanced by at least one of these:
- more explicit evidence use
- stronger compare/contrast reasoning
- more precise cause/effect or structure/function language
- more realistic data / timelines / models
- less "identify the picture" and more "justify the claim"

"Same content, slightly older wording" does **not** count as progression.

## Default Calvin output for each reviewed unit

Keep the artifact small and practical:
1. one-paragraph verdict
2. lesson-by-lesson findings
3. urgent actions now
4. rebuild plan for the unit
5. cross-grade progression note

## Default action rules

- **Keep** when the lesson has a distinct role and real value.
- **Tighten** when the lesson idea is sound but the task/explanation is too loose.
- **Retire from sequencing** when the lesson is substantially duplicated or actively weakens progression.
- **Replace** when the unit needs that slot but the current lesson cannot be salvaged quickly.
- **Escalate cross-grade concern** when the same broad topic is not meaningfully more advanced than lower grades.
