🛠️ Build and Run Your First Filter Trial

Grade 5 Engineering — Clean Water Design Lab Quick Reference
Big Idea

Version 1 testing gives engineers the evidence they need for redesign.

Key Vocabulary
  • trial: one full round of testing
  • clarity: how clear the filtered water looks
  • stability: whether the structure stayed secure
  • evidence: what the test revealed through observations or measurements
Remember This
  • Version 1 is supposed to teach you something.
  • Evidence should connect directly to the criteria.
  • Weak points help engineers plan better redesigns.
Use It to Explain
  • Start with: Version 1 did well at...
  • Add: The weakest part was...
  • Finish with: The evidence for that is...
