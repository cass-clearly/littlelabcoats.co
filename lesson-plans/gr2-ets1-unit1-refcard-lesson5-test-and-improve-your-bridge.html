🌉 Test and Improve Your Bridge

Grade 2 Engineering — Bridge Builders Quick Reference
Big Idea

Fair tests help engineers compare version 1 and version 2 honestly. Redesigns should come from evidence, not random guessing.

Key Vocabulary
  • fair test: same conditions each time
  • weak point: the part that struggled first
  • evidence: what you observed
  • redesign: an improved version
  • compare: notice which version worked better
Remember This
  • Keep the same gap and the same type of weight for both versions.
  • Name the weak point before choosing a redesign.
  • A strong explanation says exactly how the evidence changed from version 1 to version 2.
Use It to Explain
  • Start with: My bridge needs to...
  • Add: I chose this shape because...
  • Finish with: The evidence showed...
