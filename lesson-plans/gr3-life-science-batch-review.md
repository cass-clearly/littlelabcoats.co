### Rachel round-1 review notes

1. **Target:** `gr3-ls1-unit1-lesson1-what-is-a-life-cycle.html`  
   **Observation:** The lesson explains the shared pattern clearly, but the Student Worksheet model section mostly asks students to draw stages in order. It does not yet push them to label where birth, growth, reproduction, and death appear in the model, which is the core cross-organism pattern of the standard.  
   **Expected revision action:** Add a worksheet or answer-key prompt that explicitly asks students to connect their chosen organism's stages to the four shared life-cycle ideas.

2. **Target:** `gr3-ls1-unit1-lesson1-what-is-a-life-cycle.html`  
   **Observation:** The compare-two-organisms task is good, but the teacher-facing support could do more to distinguish “different stage names” from “different overall pattern.” Right now a student could complete the compare section with surface differences only.  
   **Expected revision action:** Strengthen the answer key and/or discussion guidance so adults prompt students back to the shared pattern, not just visual differences.

3. **Target:** `gr3-ls1-unit1-lesson2-different-life-cycles-same-big-pattern.html`  
   **Observation:** Lesson 2 successfully broadens the organism set, but it still reads close to a re-teach of Lesson 1 in places. The comparison lesson should lean harder into cross-organism reasoning rather than repeating the same explanatory language.  
   **Expected revision action:** Tighten Lesson 2 intro/teacher framing so it feels like an application-and-comparison lesson, not another first introduction to the concept.

4. **Target:** `gr3-ls1-unit1-lesson2-different-life-cycles-same-big-pattern.html`  
   **Observation:** The worksheet asks for one similarity and two differences, but it does not explicitly ask students to identify how reproduction appears in each compared organism. That is the part of the pattern students often skip or talk around.  
   **Expected revision action:** Add a compare prompt or answer-key note that makes students identify where reproduction shows up in the organisms they compare.

5. **Target:** `Grade 3 LS1 batch`  
   **Observation:** The unit shell is strong and family-facing, but both lessons would benefit from slightly more evidence/model language so the work feels scientific rather than only descriptive.  
   **Expected revision action:** Revise key worksheet/answer-key lines so students are prompted to use their model or comparison as evidence for the shared life-cycle pattern.

### Margaret round-1 review notes

1. **Target:** `gr3-ls1-unit1-lesson1-what-is-a-life-cycle.html`  
   **Observation:** The page structure is strong, but the materials list is long for a homeschool-family lesson and mixes required items with optional enrichment items in the same rhythm.  
   **Expected revision action:** Separate must-have materials from optional picture/reference materials so setup feels simpler and more realistic for families.

2. **Target:** `gr3-ls1-unit1-lesson1-what-is-a-life-cycle.html`  
   **Observation:** The Parent Answer Key gives good acceptable answers, but it could be more scannable for a parent who wants one sample sentence showing what a strong compare response sounds like.  
   **Expected revision action:** Add one concise model response or clearer exemplar wording for the compare prompt in the answer key.

3. **Target:** `gr3-ls1-unit1-lesson2-different-life-cycles-same-big-pattern.html`  
   **Observation:** Lesson 2 includes a lot of comparison content on one page — multiple organism cards, the pattern list, a compare table, and the worksheet compare task. The content is good, but the page risks feeling dense in print.  
   **Expected revision action:** Trim or rebalance one comparison element so the page feels more breathable without losing the core standard alignment.

4. **Target:** `gr3-ls1-unit1-lesson2-different-life-cycles-same-big-pattern.html`  
   **Observation:** The phrase “same big pattern” is clear in the title, but some worksheet directions still assume the parent/student will carry all four words in memory.  
   **Expected revision action:** Restate `birth, growth, reproduction, and death` directly in one or two worksheet directions so the task language stays self-contained.

5. **Target:** `Grade 3 LS1 batch`  
   **Observation:** The lesson pair already matches the LLC visual shell well, but the adult-support layer should do a little more of the teaching heavy lifting for homeschool use.  
   **Expected revision action:** Strengthen parent-facing answer-key guidance and exemplar phrasing before treating the stricter round-1 review as resolved.
