🌉 Build Your First Bridge

Grade 2 Engineering — Bridge Builders Quick Reference
Big Idea

Version 1 should match the plan well enough to test honestly. The point is to learn what works and what needs improvement.

Key Vocabulary
  • version 1: your first model
  • assemble: put parts together
  • fit: match the needed size
  • criteria: the success rules
  • model: something built to test an idea
Remember This
  • Build the bridge you planned, not a random new idea.
  • Check the gap before calling version 1 finished.
  • Good engineers notice one strong part and one weak part before testing.
Use It to Explain
  • Start with: My bridge needs to...
  • Add: I chose this shape because...
  • Finish with: The evidence showed...
