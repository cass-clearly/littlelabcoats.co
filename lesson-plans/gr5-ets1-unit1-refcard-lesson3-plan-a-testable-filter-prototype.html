📐 Plan a Testable Filter Prototype

Grade 5 Engineering — Clean Water Design Lab Quick Reference
Big Idea

A good prototype plan includes both a labeled design and a fair way to test it.

Key Vocabulary
  • variable: a factor that can change
  • control: something kept the same for fairness
  • fair test: a comparison that changes only what matters
  • procedure: the ordered steps of the test
Remember This
  • Labeled diagrams communicate better than vague sketches.
  • Controlled variables make evidence more trustworthy.
  • A classmate should be able to follow your plan.
Use It to Explain
  • Start with: We will keep ___ the same because...
  • Add: The variable we may change later is...
  • Finish with: This test is fair because...
