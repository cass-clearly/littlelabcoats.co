🚗 Motion Changes with Force 🚗

Physical Science — Forces & Motion | Unit 1, Lesson 2 of 6

📚 Grade: 3rd Grade ⏱️ Duration: 45–60 minutes ⚡ Subject: Physical Science
🚗➡️
A stronger push can change how far an object moves. Repeated tests help us spot the pattern.
📋 NGSS Standard:
3-PS2-1 — Plan and conduct an investigation to provide evidence of the effects of balanced and unbalanced forces on the motion of an object.

💡 The Big Idea

A force can change an object's motion. When we compare a gentle push to a stronger push, we can collect evidence that stronger and weaker forces can change how far an object moves.

🔬 The Science

Scientists look for patterns by keeping most parts of a test the same and changing just one thing. In this lesson, students keep the same object and starting point, but compare different strengths of pushes.

If a stronger push acts on an object, the motion may change more than it does with a weaker push. Repeating the test helps students gather evidence instead of guessing from only one try.

What pattern are we looking for?

When the push gets stronger, the object usually moves farther or faster. The exact numbers may change a little, but the pattern should show up across repeated trials.

💡 Key vocabulary:
force motion distance stronger force weaker force evidence compare

📦 Materials

  • 1 toy car, small block, or object that slides
  • 1 flat table or smooth floor surface
  • Masking tape for a starting line
  • Ruler or measuring tape
  • Pencil
  • Printed Student Worksheet

🔬 Let's Get Started

Step 1 — Make a Starting Line (5 min)
Use tape to mark one starting line. Put the object in the same spot every time so the test stays fair.
Step 2 — Try a Gentle Push (8 min)
Push the object gently. Measure how far it moved. Record the result.
Step 3 — Repeat the Gentle Push (5 min)
Do a second gentle push from the same starting line. Compare the two results.
Step 4 — Try a Stronger Push (8 min)
Push the same object harder from the same starting line. Measure and record how far it moved.
Step 5 — Repeat and Compare (10 min)
Do a second strong push. Then compare the gentle pushes to the strong pushes. What pattern do you notice?

💬 Discussion Questions

🗣️ Question 1: What happened when the push was stronger?Look for: The object usually moved farther or faster when the push was stronger.
🗣️ Question 2: Why did we use the same object and starting line each time?Look for: Keeping those parts the same makes the test fair, so we can compare the push strength.
🗣️ Question 3: Why do scientists repeat tests?Look for: Repeating helps us see whether the same pattern happens again and gives stronger evidence.
🗣️ Question 4: What evidence from your chart supports your claim?Look for: Students should point to measured distances or clear repeated observations.

✨ Extensions

  • 📏 Average It: Find the average distance for gentle pushes and for strong pushes.
  • 🧱 Change One Variable: Try a heavier object next. Does the same pattern still appear?
  • 🚦 Real Life: Think of a time when a stronger push or pull changed motion in real life, like opening a door or kicking a ball.

👩‍🏫 Parent/Teacher Notes

  • Fair-test reminder: Keep the object and starting line the same. The lesson works best when students clearly see that push strength is the variable changing.
  • Measurement does not need to be perfect: The goal is pattern-finding, not exact lab precision.
  • Evidence language matters: Encourage students to point to measured distances or repeated observations, not just say “because it was stronger.”
  • Assessment check: Students should be able to make a claim and back it up with data from their chart.

🔑 Parent Answer Key

Detach note: This page is for parent/teacher reference. Student wording may vary if the science idea is correct.

Part 1

Distances vary. Strong pushes should usually move the object farther than gentle pushes.

Part 2

  • Pattern claim: when the push was stronger, the object usually moved farther / moved faster.
  • Evidence to look for: measured distances or repeated observations from the test chart.
  • Best responses compare trials: for example, “both strong pushes went farther than both gentle pushes.”
  • We repeated the test because: scientists need better evidence and want to see if the same pattern happens again.

What Success Looks Like

  • Student compares stronger and weaker pushes correctly.
  • Student uses evidence from repeated tests instead of guessing.
  • Student can point to at least two trials when explaining the pattern.
  • Student understands that keeping the object and start line the same helps make a fair comparison.

Troubleshooting / Misconceptions

  • If results are messy, remind students that repeated tests matter more than one outlier.
  • If a student says “strong always means faster,” point back to the actual evidence they collected and discuss how we describe what we observed.
